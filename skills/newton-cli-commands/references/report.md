# newton report

## Purpose
Generates a textual or JSON summary describing results, recommendations, and anomalies for a given execution.

## Required Input
- `EXECUTION`: Execution identifier to summarize.

## Important Flags
- `--workspace <PATH>`: workspace containing the source artifacts (default `.`).
- `--format <text|json>`: choose output representation (default `text`).

## Example Invocation
```bash
cargo run -- report exec_789 --workspace ./workspace --format json
```

## Extra Tips
- `newton report --help` shows output format options.
- Use `--format text` for human-readable summaries and `--format json` for scripting.
- Reports require artifacts generated by a prior `run` or `step`; ensure the execution ID exists before running.
